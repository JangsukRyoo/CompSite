# CompSite

 
 개발 환경 
           
         Language: JAVA, HTML, CSS, JavaScript, Jquery

          DB   : ORACLE DB 11
          
          SERVER  : APache Tomcat 9.0

            
 기능      
 
           회원가입 /로그인/ 마이페이지 기능 
           컴퓨터 부품 소개 
           PC 추천, 원하는 부품 선택 및 즐겨찾기 기능 
           게시판 댓글 달기 / 페이징 
           IT 뉴스 크롤링
